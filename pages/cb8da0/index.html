IDEA 2024.1.1安装教程原创

# 前言

IDEA 新版本发布了,2024.1.1 这个版本是否还能白嫖？我亲测了一下，可以激活到 2025 年（实际是永久激活，只是这样会低调一些，防止被官方针对），无图无真相，下面是激活成功的截图。

目前网上有三种比较靠谱的激活方案（这三种亲测可用），小伙伴任选其一即可：

  • 第一种【推荐】：👉激活脚本 + 激活码（全自动模式），即本文教程所写，这种方法适合最新的几个版本，具体步骤跟着本文教程一步一步来，运行一下激活脚本，一分钟即可搞定，超级简单。

PS: 此方法和下面的第二种方法的补丁原理都是暴力拦截了 IDEA 的剩余使用期网络请求，实际上等于永久激活，至于激活到哪天，在之前的 2021.3.3 2021.3.2 等老版本中是可以手动配置的，比如改成激活到 2099 年。可能是太高调了，最新的版本只能显示到 2025 年, 其实效果还是一样的, 都是永久激活，可放心食用。

  • 第二种：👉破解补丁 + 激活码（需手动引用补丁），相比较于第一种无脑执行一个激活脚本，这种方式需要手动引用破解补丁，再输入激活码。此方法同样仅适用于最新的几个版本。

  • 第三种：👉 无限重置 IDEA 30 天试用期（适用于比较老的版本）。

# 下载 IDEA 安装包

访问 IDEA 官网，下载 IDEA 2024.1.1 版本的安装包，下载链接: 👉https://www.jetbrains.com/idea/download/ (opens new window)

打开页面后，点击 Download 按钮, 等待 IDEA 专业版下载完毕。

提问：电脑上已经装了其他低版本的 IDEA，需要卸载后再安装吗?
回答：不需要，本人电脑上就存在两个版本的 IDEA，没有特殊情况不需要卸载。

# 安装 IDEA 2024.1.1 版本

双击刚刚下载好的 IDEA 2024.1.1 版本安装包。弹框会提示选择安装路径，我这里直接选择的默认安装路径 C:\Program Files\JetBrains\IntelliJ IDEA 2024.1.1 , 然后点击 Next 按钮：

安装完成后，勾选 Run IntelliJ IDEA 选项，然后点击 Finish 按钮即可运行IntelliJ IDEA：

IDEA 运行成功后，会弹出如下对号框，强制用户需要先登录 JetBrains 账户才能使用：

不用管登录的事，点击 Exit 按钮退出对话框，准备开始破解激活。

# 二、Mac / Linux 系统

在 Mac / Linux 系统中，与 Windows 系统类似，需要将补丁文件夹 /jetbra 复制到一个不包含空格或中文的路径。 然后，在终端中进入 /jetbra/scripts 文件夹， 并运行 install.sh 脚本, 执行以下命令（由于需要修改环境变量，可能需要输入计算机的开机密码）：

sudo bash install.sh
1

看到提示 Done 表示激活成功。

如果提示："Operation not permitted while System Integrity Protection is engaged"，请先赋予权限，再重新执行。

chmod 755 install.sh
1

部分小伙伴 Mac/Linux 系统执行脚本遇到 "sed：RE error：illegal byte sequence"

解决方法：可以先执行下面命令，再运行脚本。

export LC_COLLATE='C'
export LC_CTYPE='C'

# 原理

执行脚本，都干了些啥？这得看是什么操作系统。

Windows 用户执行脚本后，脚本会自动在环境变量 → 用户变量下添加了 IDEA_VM_OPTIONS 变量，变量值为 /jetbra 文件夹下的 .vmoptions 参数文件绝对路径，如下图所示：

然后，脚本自动在 idea.vmoptions 文件中引用了破解补丁 ：

提示

细心的朋友可能已经注意到，这种破解方式与文章开头提到的《第二种 IDEA 破解方法》的区别在于，这种方式提供了自动化脚本，脚本省去了手动引入补丁的繁琐步骤，一切都由脚本完成。

Mac / Linux 用户在执行脚本后，脚本会自动在当前用户的环境变量文件中添加相关参数。 在 Mac / Linux 系统上，需要重新启动系统以确保环境变量生效。 您也可以自行检查，如果没有看到这些参数被自动添加，那可能意味着脚本执行并未成功。
