GitLab配置SSH密钥教程

序言

本文详细讲解如何生成SSH密钥并在GitLab上配置

2024年05月24日 @Regulus

  1. 首先点击右键，选择 Git Bash Here 打开 Git 命令行窗口：

  1. 创建密钥，在命令行窗口中输入：
ssh-keygen -t rsa -C "你的邮箱"
1

一路回车后会在账户的主目录下，如 C:\Users\用户\.ssh\ 生成两个文件：

  • id_rsa：私钥
  • id_rsa.pub：公钥
  1. 复制 id_rsa.pub公钥中的内容，在命令行窗口中输入以下命令：
cat id_rsa.pub
1

复制下图中红色框选的内容：

注意路径必须在 C:\Users\用户\.ssh 下

路径不对时，输入以下命令切换到正确的路径：

cd ~/.ssh
1
  1. 打开GitLab，点击右上角头像，选择 Settings，然后选择 SSH 密钥，将公钥内容粘贴到密钥框中：

最后点击增加密钥按钮，即可生成一条密钥记录，这样就成功在GitLab上配置了SSH密钥。
